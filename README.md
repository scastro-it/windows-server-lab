# Laboratorio de Infraestructura IT

Este repositorio documenta prácticas reales realizadas en un entorno de laboratorio
con un servidor físico HPE ProLiant DL360 Gen10.

El objetivo es aprender y ganar experiencia práctica en tareas de infraestructura IT,
especialmente sobre servidores Windows, almacenamiento y soporte técnico.

No es un proyecto de desarrollo ni de programación, sino un registro de pruebas,
configuraciones, problemas encontrados y decisiones técnicas tomadas.

## Entorno
- Servidor: HPE ProLiant DL360 Gen10
- Sistema operativo: Windows Server 2019
- Discos: 2 discos rígidos SATA
- Uso: laboratorio / pruebas (servidor fuera de producción)

## Documentación
En este repositorio se incluyen los siguientes documentos:

- **almacenamiento_y_raid.md**  
  Análisis del almacenamiento, RAID, limitaciones del hardware y alternativas evaluadas.

- **instalación_windows_server_2019.md**  
  Proceso de instalación de Windows Server, drivers y configuración inicial.

*(Se irán agregando nuevos documentos a medida que avancen las prácticas)*

## Objetivo personal
Adquirir experiencia práctica orientada a roles de:
- Soporte Técnico L2
- Infraestructura IT Junior
- Sysadmin Junior (en formación)

Todo el contenido está basado en pruebas reales realizadas sobre el servidor.
